# miet-3d-js
Учебный проект по трехмерной графике в JS.

## Папка FinalProject - итоговый проект.
### Описание проекта
Название: Учебный планировщик задач
Веб-приложение для создания, управления и отслеживания списка учебных заданий.

Из возможностей:
1. Добавление задач: Возможность добавлять новые задачи, указывая предмет (дисциплину), детальное описание, дату сдачи (дедлайн) и начальный статус ('К выполнению').
2. Отображение и фильтрация: Просмотр всех задач с возможностью фильтрации списка по предмету и по статусу выполнения. Задачи автоматически сортируются по дате сдачи.
3. Автоматическая подсветка просроченных заданий: Задачи, дедлайн которых истек и которые не имеют статус 'Выполнено', автоматически выделяются красным цветом для наглядности.
4. Изменение статуса: Легкое переключение статуса задачи между 'К выполнению' и 'Выполнено'.
5. Удаление задач: Возможность удаления ненужных задач из списка.

Есть адаптивная верстка через Flexbox

#### Технологии
- **Бэкэнд:** TypeScript, Bun (среда выполнения), Elysia (быстрый веб-фреймворк).
- **БД:** SQLite
- **Фронтэнд:** Next.js (React)

### Как запустить
Для запуска требуется установленный Bun (для сервера) и Node.js (для клиента, через npm).

Запуск Бэкенда:
1. Откройте терминал в папке `FinalProject/server`.
2. Запустите сервер:
   ```bash
   bun run src/index.ts
   ```
3. Сервер будет доступен по адресу: **http://localhost:3000**

Запуск Фронтенда:
1. Откройте НОВЫЙ терминал (не закрывайте терминал с бэкендом) в папке `FinalProject/client`.
2. Запустите фронтенд-сервер:
    ```bash
   npm run dev
   ```
3. Перейдите по адресу:
После запуска фронтенда, зайдите в браузере по адресу, который укажет Next.js в терминале: **http://localhost:3001**
